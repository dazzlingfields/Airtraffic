Notes

  • Live ADS-B data can lag or disappear when coverage is poor.
  • Some registrations return metadata even when the aircraft is not visible live.
  • The rarity score is approximate and is meant for quick context, not certification-level analysis.
  • The New Zealand register is public and updated on the CAA site; this page tries to use that data when the browser allows it.
